# 📌 Sobre o Projeto

A internet possui informação infinita.

O verdadeiro problema nunca foi falta de conteúdo.

O problema é:

* O que estudar?
* Em que ordem estudar?
* Qual vídeo escolher?
* O que fazer depois?

Eu próprio enfrentei esse problema.

Passei inúmeras horas pesquisando cursos, vídeos, artigos e tentando montar um caminho lógico de aprendizagem.

O **Roadmap Vivo** nasceu para resolver exatamente isso.

Este projeto é o meu sistema pessoal de aprendizagem, disponibilizado publicamente para qualquer pessoa que queira acompanhar a mesma jornada.

Aqui, não é necessário perder tempo pesquisando.

Tudo já foi organizado.

Você apenas entra, aprende e avança.

---

# 🎯 Objetivos

### Para mim (Curador)

* Organizar a minha própria jornada de aprendizagem.
* Atualizar continuamente o roadmap à medida que aprendo.
* Registar os melhores recursos encontrados ao longo do caminho.

### Para quem acompanha

* Poupar centenas de horas de pesquisa.
* Ter um caminho estruturado e validado.
* Aprender através dos mesmos recursos que estou a utilizar.

---

# 🧠 Filosofia

O Roadmap Vivo segue uma ideia simples:

```text
Menos pesquisa.
Mais aprendizagem.
```

Cada recurso presente aqui existe por algum motivo:

✅ Explica bem o assunto.

✅ Complementa outros tópicos.

✅ Foi utilizado ou validado durante a minha própria aprendizagem.

O roadmap está constantemente a evoluir.

Se amanhã eu encontrar um recurso melhor, o roadmap melhora.

Se eu reorganizar a ordem de estudo, o roadmap evolui.

Por isso ele é um **Roadmap Vivo**.
